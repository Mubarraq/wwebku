Siapa saya ?

Tentang saya

Saya adalah seorang mahasiswa yang sedang menempuh pendidikan di Universitas Muhammadiyah Mahakarya Aceh. Di sini, saya berfokus untuk mengembangkan kemampuan saya dalam berbagai bidang, dengan tujuan untuk terus belajar dan berkontribusi positif. Saya memiliki pemahaman dasar mengenai pengembangan web, termasuk beberapa konsep seperti HTML, CSS, dan JavaScript. Meskipun masih dalam tahap belajar, saya sangat antusias untuk mengembangkan keterampilan web saya lebih lanjut dan memanfaatkan teknologi untuk menciptakan solusi yang bermanfaat. Saya juga aktif mencari kesempatan untuk memperluas pengetahuan dan keterampilan dalam dunia teknologi dan pengembangan perangkat lunak. Dengan semangat untuk terus berkembang, saya berharap dapat memberikan kontribusi yang berarti di bidang teknologi di masa depan.
